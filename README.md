### 项目简介
本项目为2023年全国大学生数学建模竞赛C题，主要研究了商超蔬菜类商品补货和定价策略的问题，利用频繁模式分析（FP-Growth）、季节性预测模型和熵权-TOPSIS 法等机器学习方法分析。
### 技术栈
本项目主要基于 Python 构建了完整的数据处理与建模流水线：
- 数据工程与清洗: Pandas, Numpy
- 数据可视化: Matplotlib, Seaborn
- 机器学习与算法建模: Scikit-learn, SciPy
- 开发环境: Jupyter Notebook
